OVERVIEW
========

Content type selector to quickly switch from one type to another in content type
administration pages.

When editing CCK fields, the selector is built with the types where the field
really exists. This behavior has the nice effect that you can quickly check if
the field being edited is shared by simple inspection of the selector options,
and it offers a quick method to switch from one type to another to compare the
instance related settings of the field on different content types.


INSTALLATION
============

- Copy all contents of this package to your modules directory preserving
  subdirectory structure.

- Goto Administer > Site building > Modules to install this module.

DESCRIPTION
===========

The admin module provides UI improvements to the standard Drupal admin
interface. It implements some of the ideas being explored for
usability improvements in Drupal 7.


INSTALLATION
============

1. Install & enable the module.

2. Go to admin/settings/linkweights and configure links per content type.  You must have at 
   least one node for each content type you wish to configure for the module to find out 
   which other modules use hook_link
